# gcr

## Requirements

You will need:
* `gcloud` command tool
* `docker` command tool
* `curl` command

## Demos

### run-push.sh

Pushes a local Docker image into a GCR repository.

### run-pull.sh

Pulls a Docker image from a GCR repository to your local computer.
